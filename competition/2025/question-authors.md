## 📝 Question Guidelines

### Difficulty

💪 Questions should be at least **3-dot** difficulty — comparable to HSNCT/PACE NSC level. The first one or two lines should be unanswerable by baseline AI systems.

### Format

- 🧩 **Length**: ~450 characters (target range 400–700)
- **Type**: Tossups and bonuses
- **Answers**: Must match Wikipedia page titles
- **Style**: Pyramid structure — hardest clues first, easier clues toward the end

### Themed Packets

The packets are themed, and we have rounds on:

- Military history
- World literature
- Music
- Canadian history
- And more!

### What Makes a Good Adversarial Question?

**Good questions** are adversarial if they:

- Paraphrase rather than quote standard clues
- Require multi-hop or logical reasoning
- Demand commonsense or expert-level knowledge
- Include mathematical elements
- Use cross-lingual references

**Avoid** questions that:

- Use vague location descriptors
- Contain false presuppositions
- Add subjective specificity that penalizes knowledge
- Use "swerve" structures that redirect expectations mid-question

A good question is one where a knowledgeable human immediately recognizes the correct answer, even if the AI does not. See [How to Stump a Computer](/articles/stump-a-computer/) for tips.

## 📊 Scoring

Submitted questions are scored on four dimensions (25 points each):

| Criterion | Description |
|---|---|
| **Discriminability** | Distinguishes player skill levels via item response theory |
| **Difficulty** | Questions where humans outperform AI |
| **Diversity** | Varied entities, locations, and named entities across the set |
| **Quality/Quantity** | Number of questions passing editorial review |

Scoring uses [CALSCORE](https://arxiv.org/abs/2502.19684). See the [bonus scoring details](https://docs.google.com/document/d/1AvCGTBB43SV3iR_q80Y-CEzkZZoQSHD97m90Dg1u1OA/edit?usp=sharing) for bonus-specific rubrics.
